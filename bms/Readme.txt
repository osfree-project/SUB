OS/2 Base Mouse Subsystem
============================

This is open-source OS/2 Base Mouse Subsystem replacement. It is binary compatible with original OS/2 BMS.
This source code used to build - OS/2 and DOS versions of BMSCALLS.DLL and DOS Static libraries API.LIB and FAPI.LIB

See DOS/fapi and OS2/CPI/doscalls for makefiles
